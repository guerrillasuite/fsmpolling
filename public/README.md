# Public Assets

## FSM Logo Required

Please add the Free Speech Media Network logo as:
- **File name:** `fsm-logo.png`
- **Location:** This directory (`public/fsm-logo.png`)
- **Recommended size:** Height around 96-128px (will be displayed at 48px and 64px)
- **Format:** PNG with transparent background

The logo will appear in two places:
1. Top-left corner of the survey (non-clickable, 48px height)
2. Bottom of the completion screen (clickable link to freespeechmedia.live, 64px height)
